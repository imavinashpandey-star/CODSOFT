# CODSOFT Artificial Intelligence Internship

This repository contains the projects completed as part of my **CodSoft Artificial Intelligence Internship**. These projects demonstrate the fundamentals of Artificial Intelligence, Python programming, and problem-solving.

## Tasks Completed

### Task 1: Rule-Based Chatbot
A simple chatbot that responds to user queries using predefined rules and pattern matching.

**Technologies Used:**
- Python
- Conditional Statements
- Loops

### Task 2: Tic-Tac-Toe AI
An AI-powered Tic-Tac-Toe game where the computer uses the Minimax algorithm to make optimal moves.

**Technologies Used:**
- Python
- Minimax Algorithm

### Task 3: Recommendation System
A movie recommendation system that suggests movies based on the user's preferred genre.

**Technologies Used:**
- Python
- Lists
- Dictionaries
